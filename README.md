## 🛠️ Herramientas del Curso

En esta asignatura no utilizaremos instalaciones locales complejas. Todo el desarrollo se realizará en la nube a través de **GitHub Codespaces**. Las tecnologías principales que aprenderás a dominar incluyen:

* **Python 3.12+:** El lenguaje base para nuestra lógica algorítmica y cálculo computacional.
* **Git y GitHub:** Control de versiones profesional para gestionar y entregar tus evaluaciones.
* **uv:** El gestor de paquetes ultrarrápido que inicializará tus entornos de trabajo en milisegundos.
* **Marimo Notebooks:** Cuadernos interactivos y reactivos que utilizaremos para la experimentación matemática, análisis numérico y visualización geométrica pura (fractales, circuitos lógicos, etc.).

---

## 🗂️ Estructura de la Organización

Si estás navegando por la organización de GitHub del curso, notarás que consta de múltiples repositorios. Así es como funciona nuestro ecosistema:

1.  **`website` (Este repositorio):** Contiene el código fuente en Quarto de los apuntes del curso. Es de solo lectura para los estudiantes.
2.  **Repositorios de Ejercicios (`ejercicio-XX`):** Cada vez que se asigne uno o más ejercicios, recibirás una copia privada de una plantilla. Ahí es donde escribirás tu código.
3.  **Autograder:** Tus entregas serán evaluadas automáticamente mediante GitHub Actions tras cada envío (`push`), proporcionándote retroalimentación inmediata sobre la precisión de tus algoritmos.
